# Neon Hangman

A modern, cyberpunk-themed Hangman game built with vanilla HTML, CSS, and JavaScript.

## Features

*   **Sleek UI**: Dark mode with neon cyan accents and smooth animations.
*   **Interactive Gameplay**: On-screen keyboard with visual feedback for correct and incorrect guesses.
*   **Dynamic Visuals**: SVG-based Hangman drawing that updates in real-time.
*   **Responsive Design**: Looks great on different screen sizes.

## How to Play

1.  Enter your name on the welcome screen to start.
2.  The game will select a random tech-related word.
3.  Guess the word by clicking letters on the virtual keyboard.
4.  You have **6 lives**. Each wrong guess adds a part to the hangman drawing.
5.  Win by guessing the word before the drawing is complete!

## Installation

To run this game locally:

1.  Clone the repository:
    ```bash
    git clone https://github.com/ramilevi1/hangman.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd hangman
    ```
3.  Open `index.html` in your web browser.

## Technologies Used

*   HTML5
*   CSS3 (Variables, Flexbox, Animations)
*   JavaScript (ES6+)
*   Google Fonts (Outfit)
